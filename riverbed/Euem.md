# The Alluvio Ecosystem

## Observability process 
- Collect Data
- Gain Insights
- Take Action


## Aternity EUE
- Collecting and analyzing device and EUE (End User Experience) from inside End-User Devices.
    - EUE Metrics

## NetIM
- Polling the Network and Server Infrastructure.
    - Synthetics
    - SNMP/WMI
    - Configs

## NetProfiler
- Collecting and analyzing Flows from network devices throughout the Enterprise and Cloud.
    - Flows

## AppResponse
- Collecting and analyzing Packets from network devices.
    - Packets

## Aternity APM
- Collecting and analyzing transactions inside critical application
    - APM Metrics

## Shifiting Left

## Use cases

### IT Service Desk Dashboard
- Presents detailed information about device configuration, installed software, and health events.
- Service Desk Alerts + Auto Remediation.


### Digital Transformation
- The **Digital Experience Index** Dashboard provides a holistic view of the largest areas of improvement across enterprise IT, based on multiple criteria.


### Collaboration Tool Analysis
- EUEM **Collaboration Tool Library** Dashboard helps to access the performance and productivity impact of all meeting application.

### Smart Device Refresh
- EUEM helps in indentifying thw worst performing older machines and suggests lower-cost upgrades.



## KPIs for  Activites 
- Activities Response or Click-to-Render Response
    - Client Time 
    - Network Time
    - Backend Time
    - Activity Score

## KPIs for Application Performance

- Performance
    - Usage Time
    - Hang Time
    - Wait Time
    - PageLoad Time
    - Crashes
    - Page Error Rate

- User Experience Index

## KPIs for Device Performance
- Health Events: Event viewer events or device performance breaches.
- Installed Software and Its changes.
- Host and process resource usage.
- System boot times and loading time for GPOs and service components.
- Wireless connection properties.
- Custom Metrics.
- System and user attributes.
- Stability Index
- Device Score

### Activity Listener
- Sits between the application and the os to Measure times.
- Measure the times for communicating with each component of the network.
- Creates activity score for each activity (Base lining).
