## SteelHead

### The main goal of SteelHead products is to optimize the movement of datasets across Datacenters and Users securely and efficiently.

- Movement of data in and between clouds is expensive so a solution  is needed to optimize it.
- There are two ways to move data: direct connections and VPN.
- With Riverbed ==SteelHead Cloud==, we can sidestep the tradeoffs between direct connection services and VPN data transfer to the cloud.
- Riverbed helps us overcome the risks of diminished performance as you migrate to cloud with our Cloud and SaaS acceleration solutions.

### Products

- ==Riverbed SteelHead== — A WAN optimization appliance that accelerates applications, reduces data transfer, and improves network performance across distributed locations.

- ==Riverbed SteelHead Virtual== — A virtualized version of SteelHead that provides application acceleration inside virtual data center and edge environments.

- ==Riverbed SteelHead Cloud== — A cloud-based acceleration solution for IaaS platforms like AWS and Azure that speeds up cloud migrations and workload access.

- ==Riverbed SteelHead RS== — An edge data resiliency solution that keeps critical data available locally and synchronized during network disruptions.

- ==Riverbed SteelHead Mobile== — A client solution that accelerates and secures application access for remote and mobile users on any network.

- ==Riverbed SteelHead SaaS== — A performance optimization platform for SaaS applications like Microsoft 365 and Salesforce that reduces latency and improves user experience.

- ==Riverbed SteelCentral Controller== — A centralized management system for configuring, monitoring, troubleshooting, and reporting across all SteelHead deployments.

#### SteelHead can move massive data sets between data centers, cloud or edge locations to fully leverage enterprise and AI applications.
