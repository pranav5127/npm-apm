# Application Performance Monitoring (APM)

- APM is the practice of tracking key software application peformance metrics using monitoring software and telemetry data.

- It provides insights into how users experience applications and where performance gaps lie.

## Observability vs Monitoring

- APM is about monitoring application performance and system helath by capturing and displaying data which can be analyzed by various teams.

- Observability checks the system's internal state based on the data generated such logs, metrics and traces.

## Uses

- Continuos improvement

- Cloud resource utiliztion

- Application security

- Discover problems before they disrupt

- Determine the root cause of issues

## Core features

- Infrastructure monitoring
- Network monitoring
- Database monitoring
- Log monitoring
- Container monitoring
- Cloud monitoring
- Serverless monitoring
- Synthetic monitoring
- End-user monitoring
