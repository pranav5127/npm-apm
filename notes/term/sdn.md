# SDN (Software Defined Networking)

- SDN is a category of technologies that use software to manage a newtork.

- SDN software is interoperable with any router or switch.

- In networks with only physical hardware, each individual router or switch has to be configured. 

- In SDN the control plane is separated from the data plane and the actual hardware, which makes it possible to configure from a central location.

  - **Plane** is a networking term that refers to an abstract conception of where networking processes take place. The control plane directs network traffic and data plane is the actual data traversing the network.

- https://www.cloudflare.com/learning/network-layer/what-is-sdn/

## Uses 

- SDNs are widely used in large data centers.

- Data centers contain servers and networking equipment for storing, processing, and exchanging data.

- Many companies use data centers for websites, corporate data, and internal applications.

- SDN simplifies network management in data centers by controlling networks through software.

- SDN helps connect on-premise infrastructure with cloud infrastructure in hybrid cloud setups.

- Software-based integration is easier and avoids hardware compatibility issues.

- Many vendors provide both cloud services and SDN solutions for simpler hybrid cloud integration.
